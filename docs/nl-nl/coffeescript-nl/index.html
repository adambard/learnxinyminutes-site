CoffeeScript is een kleine programmeertaal die direct compileert naar JavaScript en er is geen interpretatie tijdens het uitvoeren. CoffeeScript probeert om leesbare, goed geformatteerde en goed draaiende JavaScript code te genereren, die in elke JavaScript-runtime werkt, als een opvolger van JavaScript.

Op de CoffeeScript-website, staat een volledigere tutorial voor CoffeeScript.
