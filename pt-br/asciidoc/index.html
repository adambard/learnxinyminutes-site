Onde X=AsciiDoc

Baixar o código: asciidoc.adoc

AsciiDoc é uma linguagem de marcação similar ao Markdown e pode ser usada para qualquer coisa, de livros até blogs. Criada em 2002 por Stuart Rackham, a linguagem é simples mas facilita muito a customização.

Cabeçalho do documento

Cabeçalhos são opcionais e não podem conter linhas em branco. Devem estar separados do conteúdo com pelo menos uma linha em branco.

Parágrafos

Você não precisa fazer nada especial para criar um parágrafo.

Adicione uma linha em branco entre os parágrafos para separá-los.

Para criar uma linha em branco adicione um +
e você terá uma quebra de linha!

Formatando texto

_underscore é itálico_
*asterisco é negrito*
*_você pode combinar efeitos_*
`use crase para fonte monoespaçada`
`*fonte monoespaçada em negrito*`

Listas

Para criar uma lista com marcadores use asteriscos.

* foo
* bar
* baz

Para criar uma lista numerada use pontos.

. item 1
. item 2
. item 3

Você pode criar listas dentro de listas adicionando asteriscos ou pontos extras em até 5 níveis.

* foo 1
** foo 2
*** foo 3
**** foo 4
***** foo 5

. foo 1
.. foo 2
... foo 3
.... foo 4
..... foo 5
