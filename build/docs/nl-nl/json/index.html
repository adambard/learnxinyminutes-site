Leer X in Y minuten

Waar X=json

Download de code: learnjson-nl.json

Aangezien JSON een extreem eenvoudig datauitwisselingsformaat is, zal dit waarschijnlijk de meest eenvoudige Learn X in Y Minutes ooit zijn.

Puur JSON heeft geen commentaar, maar de meeste parsers zullen commentaar in de stijl van C (//, /* */) aanvaarden. In dit voorbeeld zal alles 100% correcte JSON zijn. Gelukkig spreekt het meeste voor zichzelf.

  "heeft json booleans?": true,
  "niets": null,

  "objecten": {
    "commentaar": "De meeste structuur wordt gemaakt met objecten.",
