CoffeeScript é uma pequena linguagem que compila um-para-um para o JavaScript equivalente, e não há interpretação em tempo de execução. Como um dos sucessores de JavaScript, CoffeeScript tenta o seu melhor para exibir uma saída legível, bem-impressa e bom funcionamento dos códigos JavaScript em todo o tempo de execução JavaScript.

Veja também site do CoffeeScript, que tem um tutorial completo sobre CoffeeScript.
